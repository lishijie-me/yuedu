搭建个人博客的过程和遇到的一些问题

  • 环境准备
  • 项目配置

# 环境准备

  • NodeJS 安装

到NodeJS 官网下载安装，默认安装即可

  • 查看并更换 npm 源

    • 查看当前源
    npm get registry
    • npm 设置淘宝镜像源
    npm config set registry http://registry.npm.taobao.org
    • npm 设置官方镜像源
    npm config set registry http://www.npmjs.org

# 项目配置

# 一、创建文件夹并初始化项目

  • 创建文件夹 并 进入
mkdir REPO
cd REPO

注 ：这里的 REPO 是你的项目名称，可以随便起

  • 按默认配置初始化一个项目，此时会在当前目录下生成 package.json 文件
npm init -y
  • 在 package.json 里的 scripts 中添加如下代码，不需要修改其它代码：
{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}
  • 在项目根目录下，新建 docs 文件夹
mkdir docs
  • 新建一个 markdown 文件
echo '# Hello VuePress!' > docs/README.md
  • 启动项目，在浏览器中访问 http://localhost:8080/ 即可预览效果
npm run docs:dev
  • docs 目录下新建 .vuepress 文件夹
# 新建 .vuepress 文件夹
mkdir docs\.vuepress
  • 接着在 .vuepress 文件夹下新建 config.js 文件,这里的 config.js 便是一个 Vuepress 网站必要的配置文件，在其中添加如下代码：
module.exports = {
  base: '/REPO/',
  title: 'REPO',
  description: 'Vuepress REPO'
}

base：站点的基础路径，它的值应当总是以斜杠开始，并以斜杠结束。这里设置为 /REPO/ ，我们再次在本地运行项目，访问路径就需要变更为 http://localhost:8080/REPO/ title：网站的标题 description：网站的描述
